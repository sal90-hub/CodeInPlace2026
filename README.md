# CodeInPlace2026
A Python recipe search tool powered by an inverse index. It supports multi-word OR searches to find dishes based on cravings and moods, complete with a terminal UI.

Recipe Search Engine
A terminal-based Python application that makes finding dinner fun.

Key Features:
Smart Indexing: Automatically reads and parses all recipe files from a local directory.

Mood & Ingredient Matching: Uses an inverse index to quickly look up single or multi-word queries.

Terminal Graphics: Features custom, aligned cooking ASCII art (cooking pot, chef's knife, and dessert cake) during the user experience.
